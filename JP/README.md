# nomouse-unity (制作中...)

Unity開発の体験を飛躍的に向上させるためのCLIです。

![nomouse logo](../assets/nms-logo.png)

## プロジェクト概要


* **Status**: 内部マイルストーンやプロジェクトの現在状況を即座に表示します。
* **Gen**: 動的パラメータに対応し、テンプレートベースの高速なファイル生成を実現します。
* **Set**: テンプレートファイルに対して、暗黙的または明示的な引数を直接渡すことができます。
* **Run**: 単一コマンドでプログラム全体、または特定のテストファイルを実行します。
* **Export**: ファイル関連の設定を、ポータブルな単一ファイルに集約します。
* **Query**: CLI内でコンテキストを維持したまま、任意のLLM（大規模言語モデル）とシームレスに対話できます。
* **Doc**: 対象ドキュメントへの素早いアクセス、編集時のパラメータ更新、および新規ドキュメントの即時作成が可能です。
* **Task Queue (Emplace / Top / Pop)**: 内部プライオリティキューを使用して、一時的なメモやタスクを管理します。
* **Rename**: ファイル名と内部クラス名が一致する場合、両者を同期してリネームし、一貫性を担保します。
* **Add**: `git add` の実行と同時に、未使用のネームスペース（using句）を自動でクリーンアップする専用ワークフローです。
